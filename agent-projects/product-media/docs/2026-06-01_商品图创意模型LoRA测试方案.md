# 2026-06-01 商品图创意模型与 LoRA 测试方案

本文档用于后续系统测试“速卖通积木/玩具商品图”的创意广告效果，重点解决当前问题：整体画面缺少想象力、色彩不够奇幻卡通、模型与 LoRA 组合没有成体系比较。

## 结论先行

需要专门的“商品图创意提示词优化 skill/流程”。

当前提示词的问题不是语法，而是结构太弱：只描述“把商品放到某个场景”，缺少广告导演层面的设定，例如世界观、镜头动作、画面叙事、色彩脚本、前中后景层次、风格参考、主体戏剧性和后期质感。模型会按平铺直叙的方式完成任务，画面自然显得普通。

后续测试不应只比较模型，还要同时比较：

- prompt 框架。
- 主模型。
- LoRA 组合。
- denoise / guidance / steps / seed。
- 输入类型：原图、cutout light preview、web 参考图、VLM 事实摘要。
- 轨道：`creative_campaign` 与 `factual_product` 分开评估。

## 测试目标

第一阶段只追 `creative_campaign` 效果，不把结果当真实商品展示图。

视觉目标：

- 奇幻卡通广告感。
- 色彩更大胆：高饱和但有主次，不要灰、脏、暗。
- 镜头更有戏：低机位、近景、大透视、前景遮挡、景深、运动瞬间。
- 场景更有故事：不是“商品站在背景前”，而是“商品正在发生一件事”。
- 商品仍能识别：即使允许重绘，也要能看出原商品大类、主色、核心轮廓。

第二阶段再开 `factual_product`，做商品保真路线，不与创意广告路线混评。

## Prompt Skill 设计

后续应沉淀一个专门的 prompt 生成流程，输入为产品图与目标风格，输出为每个模型适配的 prompt。

### 输入

- product_id。
- 产品图路径。
- VLM 事实摘要：商品类型、主色、可见配件、角色数量、关键结构。
- 风格目标：奇幻卡通、电影广告、玩具海报、像素冒险、童话城堡等。
- 轨道：`creative_campaign` 或 `factual_product`。
- 模型类型：Flux Kontext、SDXL、Qwen Image Edit。

### 输出

每个产品至少输出三层提示词：

1. `creative_director_prompt`：广告导演稿，给整体图生图模型。
2. `style_prompt`：风格、色彩、材质、光影、镜头。
3. `guardrail_prompt`：避免文字、logo、包装、低质、错误风格。

### 生成规则

不要只写：

```text
put the toy in a fantasy cartoon scene
```

要写成：

```text
Create a high-budget fantasy toy-commercial key visual from the input product.
The toy becomes the hero of a miniature magical world, captured at a low macro camera angle.
Foreground has glowing dust, colored reflections and soft motion blur; midground contains the product in a dramatic action pose;
background has layered storybook architecture, oversized plants and warm window lights.
Use saturated turquoise, gold, coral and violet color scripting, glossy toy-brick material, cinematic rim light,
soft volumetric rays, shallow depth of field, playful wonder, no readable text, no packaging, no watermark.
```

## 风格脚本库

每个产品不要只套同一种背景，应按类别做风格方向。

| 商品 | 风格方向 A | 风格方向 B | 风格方向 C |
|---|---|---|---|
| 警车 | 奇幻霓虹玩具城市追逐 | 卡通夜雨英雄海报 | 童话小镇警灯救援 |
| F1 赛车 | 糖果色奇幻赛道 | 日落魔法大奖赛 | 玩具赛车盒绘风格 |
| Creeper | 像素洞穴魔法冒险 | 发光方块森林 | 卡通地牢 Boss 海报 |
| 红色跑车 | 海岸童话公路 | 未来玩具城市 | 火山熔岩跑道 |
| 巫师浴室场景 | 魔法石室舞台剧 | 温暖城堡内景 | 奇幻课堂/魔药房 |

建议色彩方向：

- `storybook warm`: 金色、珊瑚、青绿、紫罗兰。
- `neon fantasy`: 蓝紫、品红、青色、红色反光。
- `candy toy`: 明亮红、天空蓝、柠檬黄、草绿。
- `magic cave`: 翡翠绿、熔岩橙、深紫阴影。

避免默认“写实夜景 + 低饱和暗色”，这是当前画面缺乏想象力的主要原因之一。

不要一上来做全组合。按阶段缩小。

## 评分标准

每张图都按 100 分记录，不要只凭“好看”口头判断。

| 项 | 分值 | 说明 |
|---|---:|---|
| 想象力/故事性 | 25 | 是否像广告主视觉，而不是普通背景替换 |
| 奇幻卡通风格 | 20 | 色彩、造型、光影是否接近目标风格 |
| 商品识别度 | 20 | 能否认出原商品类别、主色、核心轮廓 |
| 画面完成度 | 15 | 构图、景深、光影、材质、清晰度 |
| 商业可用性 | 10 | 是否适合主图/海报/视频封面继续加工 |
| 缺陷扣分 | -10 | 伪文字、错结构、脏边、糊脸、怪异配件 |
