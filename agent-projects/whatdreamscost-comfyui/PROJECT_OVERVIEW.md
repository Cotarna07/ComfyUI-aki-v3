# WhatDreamsCost-ComfyUI 项目概览

## 项目简介
WhatDreamsCost-ComfyUI 是一个 ComfyUI 自定义节点集合，专注于视频生成和编辑，特别是 LTX 视频模型的工作流支持。该项目提供了强大的工具集，帮助创作者更高效地制作视频内容。

## 核心特性

### 1. LTX Director
完整的 LTX 2.3 时间线编辑器，提供：
- **时间线编辑**：可视化编辑视频时间线
- **关键帧设置**：精确控制关键帧位置和属性
- **音频同步**：自动同步音频和视频
- **多种模式**：支持文本到视频、图像到视频等多种模式

### 2. Multi Image Loader
智能图片加载器，支持：
- **图库浏览**：直观的图片浏览界面
- **批量加载**：同时加载多张图片
- **图片调整**：自动调整图片尺寸和格式
- **格式转换**：支持多种图片格式

### 3. LTX Sequencer
高效的视频序列器，支持：
- **序列创建**：快速创建视频序列
- **帧率设置**：灵活设置视频帧率
- **过渡效果**：添加平滑的过渡效果
- **批处理**：批量处理多个序列

### 4. Speech Length Calculator
智能语音长度计算器，支持：
- **实时计算**：实时计算语音长度
- **多种语速**：支持慢速、正常、快速三种语速
- **时间预测**：预测视频时长
- **帧数计算**：自动计算所需帧数

### 5. Load Video UI
增强的视频加载界面，支持：
- **简单界面**：直观的视频加载界面
- **实时预览**：实时预览视频内容
- **多种选项**：支持裁剪、调整大小等选项
- **拖放支持**：支持拖放文件

### 6. Load Audio UI
增强的音频加载界面，支持：
- **简单界面**：直观的音频加载界面
- **实时预览**：实时预览音频内容
- **裁剪功能**：支持音频裁剪
- **拖放支持**：支持拖放文件
