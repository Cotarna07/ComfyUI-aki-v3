# 工作流总目录

这个目录是当前工作区的唯一工作流总入口，统一采用三层结构：

- `01-shared/`：跨项目复用、已验证、可直接自动化调用的正式模板。
- `02-project/<project>/`：只服务某个项目的稳定模板与 mapping。
- `03-source/`：外部导入、供应商示例、UI 草稿、待整理生成稿与历史归档。

放置规则：

- 新导出的共享 API 工作流直接进入 `01-shared/`。
- 某个项目独占使用的模板进入 `02-project/<project>/`。
- 外部 `example_workflows/`、临时下载文件、界面另存工作流和脚本初稿先进入 `03-source/`，整理后再晋升。
- `runtime/` 不再作为正式工作流模板目录。
