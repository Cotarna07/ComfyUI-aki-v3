# 商品真实性验收门禁

## 使用方式

每个商品在出图前创建一份身份清单；每张候选图验收时复制检查表并填写 `通过 / 失败 / 无法确认`。`factual_product` 轨道存在任一失败或关键项目无法确认时，不得发布为真实商品图。

## 一票否决项

以下情况在 `factual_product` 中直接失败：

- 积木产品被重绘成真实汽车、真实建筑、普通卡通角色或其他材质。
- 增加、删除、合并、替换关键配件、轮胎、人仔、动物、武器或建筑组件。
- 为获得动作感而改变商品原始摆放关系、结构连接或核心比例。
- 将包装图上的视觉元素误作实物零件，或生成原图不存在的品牌/规格信息。
- 产品过暗、被遮挡或景深过强，导致无法验证结构。

## 输出标签

通过真实展示门禁：

```text
真实商品展示候选：已按原图核对主要结构、配件数量、颜色与材质；背景和灯光为场景化编辑。
```

仅可作创意广告图：

```text
创意广告主视觉：画面经过叙事性重绘，可能与实际商品的结构、配件或摆位不完全一致；不可用于核验实物内容。
```

未完成核验：

```text
待验收预览：尚未完成商品结构与配件对照，不可用于商品发布。
```
