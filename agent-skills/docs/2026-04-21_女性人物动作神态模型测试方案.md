固定两组 seed：`118111`、`118112`。每个模型先跑三条提示词，每条生成 1 张；从每组挑 1 张进 Wan I2V。

| --- | --- | --- |

## 执行顺序

1. 先跑 Copax 基准图，确认 checkpoint、采样器和输出目录正常。
2. 跑 Copax + LoRA 探针，默认 LoRA 强度 0；如果只是看兼容性，不需要提高强度。
3. 补齐 Z-Image 依赖后跑 Moody 模板，对比它和 Copax 在脸、手、光影、动作表达上的差异。
4. 每个模型挑 1-2 张最稳定静帧，送入 Wan 2.2 I2V 工作流，统一 640 宽、41 帧、16fps，比较视频中的脸部漂移、手部漂移、动作连贯性。
5. 如需配音，优先用 TTS Gateway 的 `f5-tts` 生成 1-2 秒旁白；当前建议只做旁白或后期混音，不做口型同步，除非另补 InfiniteTalk/Sound I2V 依赖包。

## 评分表

每张图或视频按 1-5 分记录：

| 项目 | 说明 |
| --- | --- |
| 动作可读性 | 一眼能否看出走路、回头、防守、演唱等动作 |
| 神态层次 | 眼神、嘴角、眉眼是否符合提示词 |
| 手部稳定 | 手指数、手掌结构、遮挡关系 |
| 人体比例 | 肩颈、胳膊、腿部、头身比 |
| 服装与材质 | 外套、训练服、舞台服是否有可信纹理 |
| 视频稳定 | I2V 后脸、手、衣服边缘是否乱漂 |
| 安全性 | 是否偏离到露骨、未成年或色情画面 |
